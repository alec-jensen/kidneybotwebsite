Commands

FUN:

  • meme - get a meme
  • yomama - get a yomama joke
  • dadjoke - get a dadjoke
  • dog - get a dog
  • cat - get a cat
  • duck - get a duck

MODERATION:

  • purge - purge messages
  • nick
  • kick
  • ban
  • mute
  • unmute
  • tempmute - mute user for specified amount of time
  • tempban - ban user for specified amount of time
  • changeprefix - change server prefix

OTHER:

  • help - shows help message
  • ping - show bots ping
  • devstats - view bots cpu, ram, and disk statistics
